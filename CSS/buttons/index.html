Preamble

You may use these in any website you like; they all fall under the MIT license.
If you want to disable a button, you can apply the class daxcr-disabled to your buttons through Javascript.

Button name: Nest

Class name: daxcr-button-nest


Nest No theme.


Nest - Regular Theme name: daxcr-REGULAR

Nest - Confirm Theme name: daxcr-CONFIRM

Nest - Cancel Theme name: daxcr-CANCEL

Nest - Continue Theme name: daxcr-CONTINUE

Nest - Error Theme name: daxcr-ERROR

Custom theme template:

.daxcr-button-nest.[YOUR_THEME_NAME] {
    --bg: #7e7e7e     !important;
    --shadow: #00000021     !important;
    --hover: #ffffff     !important;
    --speed: 0.2s     !important;
    --color: #313131     !important;
}

Implementation:

<span class="daxcr-button-nest [THEME_NAME]">Hello, World!</span>

Example:

<span class="daxcr-button-nest daxcr-REGULAR">Hello, World!</span>

Stylesheet:

<link rel="stylesheet" href="https://libweb.daxcr.net/CSS/buttons/nest.css"/>
